# P2 DAW - Desarrollo Web sobre Accesibilidad con Bootstrap 5

## Hugo del Castillo Deus 

Práctica individual de Desarrollo de Aplicaciones Web - DAW

## Objetivo de la Práctica

El objetivo principal de esta práctica es diseñar y desarrollar una web responsive, clara y accesible sobre la Accesibilidad Web, utilizando Bootstrap 5 como framework principal.

Se busca aplicar los principios básicos de diseño y demostrar el dominio en el uso de componentes y estructura de Bootstrap para el posicionamiento y estilo principal

## Tecnologías Usadas

La práctica se ha desarrollado utilizando las siguientes tecnologías:

  - HTML5: Estructura semántica de la web.
  - CSS3: Estilos personalizados y complementarios a Bootstrap.
  - Bootstrap 5: Framework principal para el diseño responsive, posicionamiento y uso de componentes.
    - Componentes mínimos utilizados: Carousel, Modal, Tooltip, Navegación desplegable, Grupo de Botones, Formulario, Tabla estilizada.
    - Iconos: Uso de Iconos específicos de Bootstrap.
  - Diseño: Implementación de una Guía de Estilos definida previamente.
  - Control de Versiones: Git para el desarrollo progresivo y gestión de versiones.
